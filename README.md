Portfolio site - static HTML/CSS/JS ready for GitHub Pages

How to use:
- Replace `images/profile.jpg` with your profile picture.
- Update social links in `js/main.js` by setting window.__SITE_CONFIG = { github:'https://github.com/your', linkedin:'https://www.linkedin.com/in/your', whatsapp:'https://wa.me/...' }
- Add or edit posts in `js/posts.js` (posts array). Posts are shown newest-first.
- Push this folder to the `gh-pages` branch or to the repository's root for GitHub Pages.
